# Sistema de Barbearia

Sistema simples em Java para cadastro de clientes e barbeiros, gerenciamento de serviços e histórico de atendimentos.

## Funcionalidades

- Cadastrar clientes
- Cadastrar barbeiros
- Registrar serviços prestados
- Visualizar histórico de atendimentos
- Simples e fácil de usar

## Tecnologias

- Java SE
- Eclipse IDE (para desenvolvimento)
- Git e GitHub (para versionamento)
